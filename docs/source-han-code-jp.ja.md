# Source Han Code JP（源ノ角ゴシック Code）をインストールする

[MacのFont Bookでフォントをインストールして検証する](https://support.apple.com/ja-jp/guide/font-book/fntbk1000/mac)

```session
# インストールディレクトリを確認
$ brew info font-source-han-code-jp

# インストール
$ open /opt/homebrew/Caskroom/font-source-han-code-jp/<ver>/OTC/SourceHanCodeJP.ttc
```
