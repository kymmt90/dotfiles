# 日本語入力プログラムを設定する

[Macの日本語入力ソース設定を変更する](https://support.apple.com/ja-jp/guide/japanese-input-method/jpim662a12b9/6.3/mac/26)

- 入力モードとして「Hiragana」と「Romaji」だけを有効にする
- 「Candidate window」の「Font」を「Hiragino Sans W5」に、「Font size」を「14」に変更する
- 「"¥" key generates」を「\ (Backslash)」に変更する
- 「Full-width numeral characters」を無効にする
